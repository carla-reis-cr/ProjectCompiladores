# Projeto de Compiladores

 Desenvolvido durante a disciplina com o Professor Ricardo Ferreira de Oliveira, em 01/2023. 

 Projeto desenvolvimento em duas partes, sendo elas: <br/>
   1 - Sintático e Léxico -> com base em uma gramatica desenvolvida do zero, era necessário implementar a sua analise léxica e semantica, validando com arquivos que seria lidos.<br/>
   2 - Semântico -> Com o sintático e léxico validando deveria gerar um arquivo com um código valido em uma linguagem, nesse caso optou-se em que será em Pascal.<br/>
